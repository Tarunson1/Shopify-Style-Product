# Shopify-Style-Product

A fully responsive and interactive product page, meticulously crafted to mimic the Shopify layout using only HTML, CSS, and JavaScript. Demonstrates advanced front-end capabilities without relying on any external libraries or frameworks
A modern, responsive product page inspired by Shopify, featuring interactive galleries, variant selectors, modals, bundles, and more.

## Repository

- **Owner:** https://github.com/Tarunson1
- **Repo:** https://github.com/Tarunson1/Shopify-Style-Product

## Features

- **Product Gallery**: Scrollable thumbnails and a zoomable main image.
- **Variant Selection**: Choose color and size, with persistent selection using localStorage.
- **Compare Colours Modal**: Select and compare multiple color swatches in a modal.
- **Size Chart Modal**: View a detailed size chart in a modal.
- **Pair Well With**: Horizontally scrollable carousel of related accessories.
- **Bundle Section**: Select products to bundle, see live total, and add bundle to cart with confirmation modal.
- **Tabbed Info Section**: Switch between Description, Product Information, and Shipping Details.
- **Related Products**: Grid of related products with badges.
- **Responsive Design**: Looks great on desktop and mobile.
- **Customizable Theme**: Easily adjust colors and backgrounds via CSS.

## File Structure

```

README.MD

  styles.css
  theme-overrides.css

  scripts.js
```

- `index.html`: Main HTML structure.
- `css/styles.css`: Core styles for layout and components.
- `css/theme-overrides.css`: Theme and background overrides.
- `js/scripts.js`: All interactive JavaScript logic.

## Customization

- **Colors & Fonts**: Edit `css/theme-overrides.css` and `css/styles.css`.
- **Images**: Replace placeholder image URLs in `index.html` with your own product images.
- **Product Data**: Update product names, prices, and details directly in `index.html`.

## Accessibility

- All interactive elements are keyboard accessible.
- Modals can be closed with the Escape key or by clicking outside.
